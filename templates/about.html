About This Project

Learn more about the Weather Visualization project.

What is this project?

This project is a weather visualization tool that provides real-time weather data for cities around the world. It uses APIs to fetch live weather data and displays it on an interactive map. Users can also view historical weather trends for several major Indian cities. Recent weather for any city in the world can also be viewed by entering the city name and the number of past hours, using various plots. The user can also look at the weather of nearby cities. The project is built using Flask, Plotly, and SQLite, making it a powerful tool for weather enthusiasts and researchers,as well as the general public.

Features

  • Real-time weather data visualization
  • Interactive map with weather details for multiple cities
  • Historical weather data analysis

Technologies Used

  • Flask (Python) for the backend
  • Plotly for interactive visualizations
  • SQLite for data storage
  • HTML, CSS, and Bootstrap for the frontend
